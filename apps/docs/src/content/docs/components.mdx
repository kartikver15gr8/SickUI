## Quick Install

Add any component to your project with a single command:

```bash
npx @sickui/cli add [component-name]
```

## Available Components

### Badge

A versatile badge component for displaying status, labels, or categories.

**Installation:**

```bash
npx @sickui/cli add badge
```

**Dependencies:**

- `class-variance-authority`
- `clsx`
- `tailwind-merge`

[View Badge Documentation →](/docs/components/badge)

### Button

Displays a button or a component that looks like a button.

**Installation:**

```bash
npx @sickui/cli add button
```

**Dependencies:**

- `@radix-ui/react-slot`
- `class-variance-authority`
- `clsx`
- `tailwind-merge`

[View Button Documentation →](/docs/components/button)

### Calendar

A beautiful, animated calendar component for date selection with Framer Motion animations.

**Installation:**

```bash
npx @sickui/cli add calendar
```

**Dependencies:**

- `framer-motion`
- `@emotion/is-prop-valid`
- `lucide-react`
- `class-variance-authority`
- `clsx`
- `tailwind-merge`

[View Calendar Documentation →](/docs/components/calendar)

### Skeleton

Loading placeholder components with pulse and shimmer animations.

**Installation:**

```bash
npx @sickui/cli add skeleton
```

**Dependencies:**

- `class-variance-authority`
- `clsx`
- `tailwind-merge`

[View Skeleton Documentation →](/docs/components/skeleton)
